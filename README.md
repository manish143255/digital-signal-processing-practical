# Digital Signal Processing Practical (MATLAB)

This repository contains MATLAB programs for DSP practical experiments for University of Delhi students.

## Experiments
1. Basic Discrete-Time Signals
2. Time Shifting
3. Time Scaling
4. Time Reversal
5. Down Sampling
6. Up Sampling
7. Addition of Signals
8. Subtraction of Signals
9. Multiplication of Signals
10. Periodic Signals
11. Aperiodic Signals

## Software Used
- MATLAB R2025b
